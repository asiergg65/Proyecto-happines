 HAPPINESS&Co - Agenda Cultural Gijón
HAPPINESS&Co es una plataforma integral diseñada para gestionar y consultar la agenda cultural de Gijón. El proyecto cubre desde el diseño de interfaz hasta la gestión de datos y lógica de servidor.
 
 Lo que hace el proyecto
El sistema permite a los usuarios descubrir eventos (música, teatro, arte) y a los administradores gestionar toda la información mediante una aplicación Java conectada a una base de datos.

Tecnologías utilizadas
Web (Front-end): HTML5 y Tailwind CSS (Diseño moderno y responsive).

Gestión (Back-end): Java (Programación Orientada a Objetos).

Base de Datos: MySQL (Relaciones entre usuarios, eventos y favoritos).

Sindicación: RSS (XML) para suscripción de noticias.

Diseño: Figma (Prototipado y esquemas visuales).

Estructura del Proyecto
Web: Interfaz de usuario con secciones de Inicio, Próximos Eventos, Historial y Contacto.

Base de Datos: 5 tablas interconectadas (Usuarios, Eventos, Galerías, Imágenes y Favoritos) con Vistas SQL optimizadas.

App Java: Sistema de gestión por consola con funciones CRUD:

Añadir/Eliminar Usuarios.

Gestionar Eventos y Galerías de fotos.

Controlar sistema de Favoritos.

Diseño Figma: Planos y prototipo interactivo de la experiencia de usuario.
